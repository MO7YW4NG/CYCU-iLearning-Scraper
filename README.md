## 介紹
此程式協助你一鍵下載 iLearning 平台之當學期教材 (含 pdf 和 pptx)。
## 使用方法

1. [下載 msi 安裝檔](https://github.com/MO7YW4NG/CYCU-iLearning-Scraper/releases)
2. 安裝後生成 `CYCU-iLearning-Scraper` 資料夾
3. 開啟 `app.exe` 後登入即啟用!
